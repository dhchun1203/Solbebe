# 로고 이미지 안내

이 폴더에 "Solbebe" 로고 이미지를 저장해주세요.

## 추천 이미지 생성 도구

1. **Canva** (https://www.canva.com)
   - 귀여운 폰트와 디자인 요소 제공
   - "Solbebe" 텍스트로 로고 디자인 가능

2. **Figma** (https://www.figma.com)
   - 무료 디자인 도구
   - 커스텀 로고 제작 가능

3. **Adobe Express** (https://www.adobe.com/express)
   - 로고 템플릿 제공
   - 간단한 로고 제작 가능

4. **AI 이미지 생성 도구**
   - DALL-E, Midjourney 등으로 "귀여운 Solbebe 로고" 생성

## 이미지 요구사항

- 파일명: `logo.png` 또는 `logo.svg`
- 권장 크기: 최소 200x60px
- 배경: 투명 또는 흰색
- 형식: PNG (투명 배경) 또는 SVG

## 사용 방법

1. 이미지를 `src/assets/images/` 폴더에 저장
2. 파일명을 `logo.png` 또는 `logo.svg`로 지정
3. Header 컴포넌트가 자동으로 이미지를 사용합니다
