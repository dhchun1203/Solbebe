# Public 폴더

이 폴더에 정적 파일(이미지, 아이콘 등)을 저장하세요.

## 로고 이미지

`logo.png` 또는 `logo.svg` 파일을 이 폴더에 저장하면 자동으로 로고로 사용됩니다.

### 로고 생성 방법

1. **온라인 로고 생성 도구 사용:**
   - Canva (https://www.canva.com) - "Solbebe" 텍스트로 귀여운 로고 디자인
   - LogoMaker (https://www.logomaker.com)
   - Hatchful (https://www.shopify.com/tools/logo-maker)

2. **AI 이미지 생성 도구:**
   - DALL-E, Midjourney 등에 "귀여운 Solbebe 로고, 아기 의류 쇼핑몰, 파스텔 색상" 프롬프트 입력

3. **디자인 도구:**
   - Figma, Adobe Illustrator 등으로 직접 제작

### 이미지 요구사항

- 파일명: `logo.png` 또는 `logo.svg`
- 권장 크기: 최소 200x60px (비율 유지)
- 배경: 투명 (PNG) 또는 흰색
- 형식: PNG (투명 배경 권장) 또는 SVG

### 사용 예시

로고 이미지를 `public/logo.png`로 저장하면 자동으로 헤더에 표시됩니다.
